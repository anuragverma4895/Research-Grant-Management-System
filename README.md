# Research Grant Management System

A **web-based Research Grant Management System** designed to streamline the process of submitting, reviewing, and managing research grant applications.  
The system provides separate interfaces for **applicants** and **administrators**, ensuring efficient evaluation and transparent status tracking.

## 🚀 Features

- Research grant application submission
- Admin dashboard for managing applications
- Grant valuation and evaluation panel
- Application status update functionality
- MySQL database integration
- Responsive UI using CSS and JavaScript

---

## 🧑‍💻 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Hosting: InfinityFree

---

## ⚙️ How It Works

1. Users submit research grant applications through the application form.
2. Admin reviews submitted applications via the dashboard.
3. Applications are evaluated in the valuation panel.
4. Admin updates application status (Approved / Rejected / Pending).
5. All data is stored securely in a MySQL database.
